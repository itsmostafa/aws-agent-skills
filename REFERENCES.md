# AWS Documentation References

Quick reference for official AWS documentation URLs used across all skills.

## Documentation URL Patterns

```
Developer Guide: https://docs.aws.amazon.com/<service>/latest/dg/
User Guide:      https://docs.aws.amazon.com/<service>/latest/userguide/
API Reference:   https://docs.aws.amazon.com/<service>/latest/APIReference/
CLI Reference:   https://docs.aws.amazon.com/cli/latest/reference/<service>/
```

## RSS Feeds for Updates

Each service has a documentation history RSS feed:

```
https://docs.aws.amazon.com/<service>/latest/<guide>/doc-history.rss
```

Examples:
- Lambda: `https://docs.aws.amazon.com/lambda/latest/dg/doc-history.rss`
- DynamoDB: `https://docs.aws.amazon.com/amazondynamodb/latest/developerguide/doc-history.rss`
- S3: `https://docs.aws.amazon.com/AmazonS3/latest/userguide/doc-history.rss`
